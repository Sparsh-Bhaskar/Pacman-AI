Pac-Man AI Pathfinding

This project demonstrates how Pac-Man is controlled by the player, while 3 ghosts chase him using the BFS pathfinding algorithm.

Instructions to Run

1. Install Python & Pygame
2. Open terminal in this folder
3. Run: python main.py

Key Features

  • Maze generated via 2D array
  • Ghosts use BFS to chase Pac-Man
  • Responsive keyboard controls for Pac-Man

Made by using Python & Pygame
